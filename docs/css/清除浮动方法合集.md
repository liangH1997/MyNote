### overflow
给父元素添加声明`overflow: hidden;`(触发一个[BFC](/css/BFC))

!> 弊端：`overflow: hidden;` 会隐藏掉定位在当前元素之外的内容

### 空div
在浮动元素下方添加`空div`，并给这个div添加声明如下
``` css
div{
  clear: both;
  height: 0;
  overflow: hidden;
}
```

!> 弊端：会出现很多没有意义的空盒子，造成代码的冗余

### 万能清除法
``` css
选择符:after{
  content: "";
  clear: both;
  display: block;
  height: 0;
  overflow: hidden;
  visibility: hidden;
}
```
